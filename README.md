# 🏠 VD Assistant: Your AI-Powered Interior Architect

**Transform empty rooms into inspiring homes with AI-powered interior design.** VD Assistant bridges the gap between imagination and reality by using a cutting-edge "AI Brain + Rendering Engine" workflow. Upload a photo and get instant redesigned versions in multiple high-fidelity styles.


---

## ✨ Features

* **📸 Room Upload & Analysis**: Intuitive drag-and-drop interface to analyze room dimensions and lighting.
* **🎨 6+ Curated Design Styles**: Choose from Modern, Minimalist, Scandinavian, Industrial, Luxury, or Traditional.
* **🖼️ High-Fidelity Rendering**: Powered by **Stable Diffusion XL**, generating 8K photorealistic architectural shots.
* **💬 AI Design Co-Pilot**: Real-time design help and layout tips powered by a locally hosted **Llama 3.2** via Ollama.
* **🛠️ Smart Recommendations**: Get tailored color palettes and furniture suggestions based on the generated design.
* **📱 Fully Responsive**: A beautiful Neumorphic UI that works seamlessly on desktop and mobile.
